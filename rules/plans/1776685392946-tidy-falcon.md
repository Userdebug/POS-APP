# Plan: Complete Rewrite of Tcollecte Table Implementation

## Executive Summary

Tcollecte is the central daily tracking table storing per-category daily aggregates (SI, Achats, CA, SF, ENV, Theoretical Sales, Margin, Closure status). Current implementation is **fragmented across 4+ services**, uses **incorrect stock reconstruction logic** (relies on movement transfers rather than actual product counts), has **duplicate queries**, **inconsistent naming**, and **legacy schema artifacts**. This plan outlines a clean, testable rewrite with a single repository responsible for all Tcollecte operations, correct SI/SF computation, and data integrity guarantees.

## Desired State & Principles

### Core Principles
1. **Single Source of Truth** — One repository (`DailyTrackingRepository`) owns ALL Tcollecte operations.
2. **Explicit SI Snapshots** — Store beginning-of-day stock values per category/day by capturing `produits` totals at day initialization. Do NOT reconstruct from movements.
3. **Derived SF Computed On-Demand** — Ending stock is simply current `produits` totals; don't store separately, compute when needed via `get_current_sf_values()`.
4. **Clear Data Model** — Use dataclasses (`DailyTrack`, `DailySummary`) with explicit field names.
5. **Atomic Operations** — All multi-row writes use transactions.
6. **Separation of Concerns** — Repository = SQL only; Service = business logic; UI = presentation.
7. **Backward Compatible** — Keep existing column names in DB, but clean API surface.

**Purpose**: `Tsf` stores daily per-category denormalized metrics for reporting. Values:
- `si_ttc`, `achats_ttc`, `ca_ttc`, `env_ttc` — copied from `Tcollecte`
- `sf_ttc` — recomputed from current `produits` stock (ensures accurate ending inventory snapshot)
- `vente_theorique_ttc`, `marge_ttc`, `marge_percent` — derived
- `is_closed` — mirrors `Tcollecte.cloturee`

```python
def migrate_tcollecte_data(db):
    """One-time migration to clean Tcollecte and initialize Tsf."""
    with db._connect() as conn:
        # STEP 1: Preserve CA from ca_temporaire
        conn.execute("""
            UPDATE Tcollecte 
            SET ca = ca_temporaire 
            WHERE ca = 0 AND ca_temporaire > 0
        """)
        conn.execute("UPDATE Tcollecte SET ca_temporaire = 0")
        
        # STEP 2: Ensure all days have rows for all OW categories
        days = [r['jour'] for r in conn.execute("SELECT DISTINCT jour FROM Tcollecte").fetchall()]
        ow_categories = [r['nom'] for r in conn.execute("""
            SELECT c.nom FROM categories c
            JOIN categories parent ON c.parent_id = parent.id
            WHERE parent.nom = 'Catégorie 1 - OW (Owners)'
        """).fetchall()]
        cat_id_map = {row['nom']: row['id'] for row in conn.execute("SELECT id, nom FROM categories").fetchall()}
        
        for jour in days:
            existing = [r['categorie'] for r in conn.execute("""
                SELECT c.nom AS categorie FROM Tcollecte t
                JOIN categories c ON t.categorie_id = c.id
                WHERE t.jour = ?
            """, (jour,)).fetchall()]
            for cat_name in ow_categories:
                if cat_name not in existing:
                    cat_id = cat_id_map.get(cat_name)
                    if cat_id:
                        conn.execute("""
                            INSERT OR IGNORE INTO Tcollecte 
                            (jour, categorie_id, si, achats, ca, ca_temporaire, sf, env, vente_theorique, marge, cloturee)
                            VALUES (?, ?, 0, 0, 0, 0, 0, 0, 0, 0, 0)
                        """, (jour, cat_id))
        
        # STEP 3: Remove duplicate rows (keep lowest id)
        conn.execute("""
            DELETE FROM Tcollecte
            WHERE id NOT IN (
                SELECT MIN(id) FROM Tcollecte
                GROUP BY jour, categorie_id
            )
        """)
        
        # STEP 4: Recompute derived fields once
        rows = conn.execute("""
            SELECT id, si, achats, sf, env, ca, cloturee 
            FROM Tcollecte
        """).fetchall()
        for row in rows:
            si = row['si'] or 0
            achats = row['achats'] or 0
            sf = row['sf'] or 0
            env = row['env'] or 0
            ca = row['ca'] or 0
            vente_theo = max(0, si + achats - sf - env)
            marge = ca - vente_theo
            conn.execute("""
                UPDATE Tcollecte 
                SET vente_theorique = ?, marge = ?
                WHERE id = ?
            """, (vente_theo, marge, row['id']))
        
        # STEP 5: Drop legacy tables
        conn.execute("DROP TABLE IF EXISTS suivi_journalier_categories")
        conn.execute("DROP TABLE IF EXISTS suivi_formulaire_journalier")
        
        # STEP 6: Create Tsf table and populate initial snapshot
        # Create table (SQL from 1.3). This will be idempotent.
        conn.execute("""
            CREATE TABLE IF NOT EXISTS Tsf (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                jour TEXT NOT NULL,
                categorie_id INTEGER NOT NULL,
                si_ttc INTEGER NOT NULL DEFAULT 0,
                achats_ttc INTEGER NOT NULL DEFAULT 0,
                ca_ttc INTEGER NOT NULL DEFAULT 0,
                env_ttc INTEGER NOT NULL DEFAULT 0,
                sf_ttc INTEGER NOT NULL DEFAULT 0,
                vente_theorique_ttc INTEGER NOT NULL DEFAULT 0,
                marge_ttc INTEGER NOT NULL DEFAULT 0,
                marge_percent REAL NOT NULL DEFAULT 0.0,
                is_closed INTEGER NOT NULL DEFAULT 0,
                refreshed_at TEXT NOT NULL DEFAULT (datetime('now')),
                UNIQUE(jour, categorie_id),
                FOREIGN KEY (categorie_id) REFERENCES categories(id) ON DELETE CASCADE
            )
        """)
        # Create indexes
        conn.execute("CREATE INDEX IF NOT EXISTS idx_tsf_jour ON Tsf(jour)")
        conn.execute("CREATE INDEX IF NOT EXISTS idx_tsf_jour_cat ON Tsf(jour, categorie_id)")
        conn.execute("CREATE INDEX IF NOT EXISTS idx_tsf_closed ON Tsf(is_closed, jour)")
        
        # Populate Tsf for all historical dates from cleaned Tcollecte
        # Reuse logic similar to refresh_tsf but inline for migration
        tcollecte_rows = conn.execute("""
            SELECT t.jour, t.categorie_id, c.nom AS categorie,
                   t.si, t.achats, t.ca, t.ca_temporaire, t.env, t.cloturee
            FROM Tcollecte t
            JOIN categories c ON t.categorie_id = c.id
            ORDER BY t.jour, c.nom
        """).fetchall()
        
        if tcollecte_rows:
            # Compute current SF snapshot per category (from current produits)
            sf_rows = conn.execute("""
                SELECT c.nom AS categorie, SUM((p.stock_boutique + p.stock_reserve) * p.pa) AS sf_ttc
                FROM produits p
                JOIN categories c ON p.categorie_id = c.id
                GROUP BY c.nom
            """).fetchall()
            sf_map = {str(r['categorie']): int(r['sf_ttc'] or 0) for r in sf_rows}
            
            tsf_inserts = []
            for r in tcollecte_rows:
                jour = r['jour']
                categorie_id = r['categorie_id']
                categorie = str(r['categorie'])
                is_closed = r['cloturee'] == 1
                si_ttc = int(r['si'] or 0)
                achats_ttc = int(r['achats'] or 0)
                env_ttc = int(r['env'] or 0)
                ca_ttc = int(r['ca'] if is_closed else r['ca_temporaire'] or 0)
                sf_ttc = sf_map.get(categorie, 0)
                vente_theo_ttc = max(0, si_ttc + achats_ttc - sf_ttc - env_ttc)
                marge_ttc = ca_ttc - vente_theo_ttc
                marge_pct = (marge_ttc / ca_ttc * 100.0) if ca_ttc > 0 else 0.0
                tsf_inserts.append((
                    jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                    sf_ttc, vente_theo_ttc, marge_ttc, marge_pct, int(is_closed)
                ))
            
            conn.executemany("""
                INSERT INTO Tsf (
                    jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                    sf_ttc, vente_theorique_ttc, marge_ttc, marge_percent, is_closed, refreshed_at
                ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, datetime('now'))
                ON CONFLICT(jour, categorie_id) DO UPDATE SET
                    si_ttc = excluded.si_ttc,
                    achats_ttc = excluded.achats_ttc,
                    ca_ttc = excluded.ca_ttc,
                    env_ttc = excluded.env_ttc,
                    sf_ttc = excluded.sf_ttc,
                    vente_theorique_ttc = excluded.vente_theorique_ttc,
                    marge_ttc = excluded.marge_ttc,
                    marge_percent = excluded.marge_percent,
                    is_closed = excluded.is_closed,
                    refreshed_at = datetime('now')
            """, tsf_inserts)
        
        conn.commit()
```

from dataclasses import dataclass
from typing import List, Optional

@dataclass(frozen=True, slots=True)
class DailyTrack:
    jour: str                    # YYYY-MM-DD
    category_id: int
    category_name: str
    
    # Stock values (PA × quantity in Ar)
    stock_initial: int           # SI — beginning-of-day stock value (TTC)
    purchases: int               # Achats — sum of purchase receptions (TTC)
    stock_final: int             # SF — current produits stock value (TTC), end-of-day snapshot
    removed_value: int           # ENV — sum of removed/damaged products (TTC)
    
    # Revenue
    ca_live: int                 # Real-time CA from sales (open day only)
    ca_final: int                # Final CA after closure (set at close)
    
    # State
    is_closed: bool
    updated_at: str
    
    @property
    def ca(self) -> int:
        """Current CA depending on closure state."""
        return self.ca_final if self.is_closed else self.ca_live
    
    @property
    def sales_theoretical(self) -> int:
        """Theoretical sales: SI + Achats - SF - ENV (never negative)."""
        return max(0, self.stock_initial + self.purchases - self.stock_final - self.removed_value)
    
    @property
    def margin(self) -> int:
        """Margin: CA - theoretical_sales."""
        return self.ca - self.sales_theoretical
    
    @property
    def margin_percent(self) -> Optional[float]:
        """Margin percentage."""
        theo = self.sales_theoretical
        if theo > 0:
            return (self.margin / theo) * 100.0
        return 0.0 if self.ca == 0 else None

    def __init__(self, connect, today_iso, day_bounds, resolve_category_names):

    def list_tracks_range(self, start: str, end: str) -> List[DailyTrack]: ...
    def get_open_days(self) -> List[str]: ...
    def is_closed(self, jour: str) -> bool: ...
    def get_last_closed_date(self) -> Optional[str]: ...
    def get_category_sf(self, jour: str) -> dict[str, int]:
        """Get SF values per category for a day (for J+1 init)."""
        ...
    
    # === Write (atomic per-day operations) ===
    def initialize_day(self, jour: str, carry_si_from: Optional[str] = None) -> None:
        """Create Tcollecte rows for all OW categories for given day.
        SI = yesterday's SF from Tcollecte if carry_si_from provided; else compute from current produits (first day).
        All other fields zero. Ensures idempotency (no duplicates)."""
    
    def update_live_ca(self, jour: str, increments: dict[str, int]) -> None:
        """Increment ca_live (ca_temporaire in DB) by amounts per category (from sales). Atomic."""

**Implementation Notes**:
- All SQL queries parameterized, no string formatting.
- Transaction boundaries: `initialize_day`, `update_live_ca`, `update_purchases`, `close_day` all run in single transaction per day.
- Use `INSERT ... ON CONFLICT(jour, categorie_id) DO UPDATE` for upserts.
- Helper `_get_category_id_map()` to resolve category names → IDs in bulk.
- `update_stock_metrics()`:
  - SF: `SELECT c.nom, SUM((p.stock_boutique + p.stock_reserve) * p.pa) FROM produits p JOIN categories c ON ... GROUP BY c.nom`
  - ENV: `SELECT categorie, SUM(valeur) FROM historique_produits_enleves WHERE jour = ? GROUP BY categorie`
- `update_live_ca()`: Given dict `{category: amount}`, for each category:
  - Read current `ca_live`
  - Compute `new = current + increment`
  - Upsert (or separate SELECT then UPDATE for clarity)
  - **Note**: Do NOT touch `ca` (final) here — only `ca_live`.

### Phase 3: Reporting Infrastructure — Intermediate Table (2-3 hours)

**Objective**: Create `reporting_daily_category_metrics` table and a sync mechanism to populate it from Tcollecte. This table serves `AnalyseJournaliereService` (SF/NFR reports) with pre-aggregated, denormalized data.

#### 3.1 Reporting Table Schema
Already defined in Phase 1.2. It stores:
- **Core metrics**: `si`, `achats`, `sf`, `env`, `ca_final`, `ca_live`
- **Derived**: `vente_theorique`, `marge`, `marge_percent`
- **Metadata**: `is_closed` (copied from Tcollecte's `cloturee`), `updated_at`

**Why separate?**
1. Reporting queries often need date ranges and SUM aggregations across days.
2. If we compute from Tcollecte on-the-fly with complex joins, it's slow.
3. This table can be refreshed nightly or on-demand after hours.
4. Decouples report schema from operational schema — we can change Tcollecte without breaking reports.

#### 3.2 Sync Service Method — refresh_tsf()

Add to `DailyTrackingRepository`:

```python
def refresh_tsf(self, start_date: str, end_date: str) -> None:
    """
    Populate Tsf table for date range [start_date, end_date].
    Reads from Tcollecte (operational) and computes sf_ttc from current produits stock.
    Recalculates all derived fields. Idempotent (UPSERT).
    """
    with self._connect() as conn:
        # Get all Tcollecte rows in range with category names
        rows = conn.execute("""
            SELECT 
                t.jour, t.categorie_id, c.nom AS categorie,
                t.si, t.achats, t.ca, t.ca_temporaire, t.env, t.cloturee
            FROM Tcollecte t
            JOIN categories c ON t.categorie_id = c.id
            WHERE t.jour >= ? AND t.jour <= ?
        """, (start_date, end_date)).fetchall()
        
        if not rows:
            return
        
        # Compute current SF snapshot per category from produits (live stock counts)
        sf_rows = conn.execute("""
            SELECT c.nom AS categorie, SUM((p.stock_boutique + p.stock_reserve) * p.pa) AS sf_ttc
            FROM produits p
            JOIN categories c ON p.categorie_id = c.id
            GROUP BY c.nom
        """).fetchall()
        sf_map = {str(r['categorie']): int(r['sf_ttc'] or 0) for r in sf_rows}
        
        # Build Tsf data rows
        tsf_data = []
        for r in rows:
            jour = r['jour']
            categorie_id = r['categorie_id']
            categorie = str(r['categorie'])
            is_closed = r['cloturee'] == 1
            
            si_ttc = int(r['si'] or 0)
            achats_ttc = int(r['achats'] or 0)
            env_ttc = int(r['env'] or 0)
            # Use ca if closed, else ca_temporaire (live)
            ca_ttc = int(r['ca'] if is_closed else r['ca_temporaire'] or 0)
            sf_ttc = sf_map.get(categorie, 0)
            
            # Derived
            vente_theo_ttc = max(0, si_ttc + achats_ttc - sf_ttc - env_ttc)
            marge_ttc = ca_ttc - vente_theo_ttc
            marge_pct = (marge_ttc / vente_theo_ttc * 100.0) if vente_theo_ttc > 0 else 0.0
            
            tsf_data.append((
                jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                sf_ttc, vente_theo_ttc, marge_ttc, marge_pct, int(is_closed)
            ))
        
        # Replace existing rows for range (ensures Tsf is fresh)
        conn.execute("DELETE FROM Tsf WHERE jour >= ? AND jour <= ?", (start_date, end_date))
        conn.executemany("""
            INSERT INTO Tsf (
                jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                sf_ttc, vente_theorique_ttc, marge_ttc, marge_percent, is_closed, refreshed_at
            ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, datetime('now'))
        """, tsf_data)
        conn.commit()
```

**When to call `refresh_tsf()`**:
- After any Tcollecte update that affects reports: `update_live_ca()`, `update_purchases()`, `update_stock_metrics()`, `set_final_ca()`, `close_day()`.
- On-demand when user opens SF/NFR report dialogs (refresh the requested range).
- Nightly batch for all recent days (e.g., last 30 days) to keep SF current.
- Note: `refresh_tsf()` performs its own DELETE+INSERT for the range; safe to call frequently.

```python
def get_sf_report(self, date_debut: str, date_fin: str) -> List[dict]:
    """
    SF Report: per-category metrics over date range.
    Returns: {sous_categorie, si_ttc, achats_ttc, sf_ttc, demarque_ttc, vente_theo_ttc, marge_ttc, marge_percent}
    
    Semantics:
    - si_ttc: Stock Initial at start of first day in range (from first day's Tcollecte.si)
    - achats_ttc: SUM of purchases across all days in range
    - sf_ttc: Stock Final at end of last day in range (from last day's Tcollecte.sf, but we take max date's sf)
    - demarque_ttc: SUM of removed value across range
    - vente_theo_ttc: computed = si_ttc + achats_ttc - sf_ttc - demarque_ttc
    - marge_ttc: SUM(ca_ttc over range) - vente_theo_ttc
    - marge_percent: (marge_ttc / vente_theo_ttc) × 100 if theoretical > 0 else 0
    """
    with self.repo._connect() as conn:
        # First determine first and last date per category within range
        bounds = conn.execute("""
            SELECT 
                c.nom AS categorie,
                MIN(t.jour) AS first_jour,
                MAX(t.jour) AS last_jour
            FROM Tsf t
            JOIN categories c ON t.categorie_id = c.id
            WHERE t.jour >= ? AND t.jour <= ?
            GROUP BY c.nom
        """, (date_debut, date_fin)).fetchall()
        
        result = []
        for b in bounds:
            cat = str(b['categorie'])
            first_jour = b['first_jour']
            last_jour = b['last_jour']
            
            # Get SI from first day
            si_row = conn.execute("""
                SELECT si_ttc FROM Tsf
                WHERE jour = ? AND categorie_id = (
                    SELECT id FROM categories WHERE nom = ?
                )
            """, (first_jour, cat)).fetchone()
            si_ttc = int(si_row['si_ttc'] or 0) if si_row else 0
            
            # Get SF from last day
            sf_row = conn.execute("""
                SELECT sf_ttc FROM Tsf
                WHERE jour = ? AND categorie_id = (
                    SELECT id FROM categories WHERE nom = ?
                )
            """, (last_jour, cat)).fetchone()
            sf_ttc = int(sf_row['sf_ttc'] or 0) if sf_row else 0
            
            # Get sums across full range
            agg = conn.execute("""
                SELECT 
                    SUM(achats_ttc) AS sum_achats,
                    SUM(ca_ttc) AS sum_ca,
                    SUM(env_ttc) AS sum_env
                FROM Tsf
                WHERE jour >= ? AND jour <= ? AND categorie_id = (
                    SELECT id FROM categories WHERE nom = ?
                )
            """, (date_debut, date_fin, cat)).fetchone()
            
            achats_ttc = int(agg['sum_achats'] or 0)
            ca_ttc = int(agg['sum_ca'] or 0)
            demarque_ttc = int(agg['sum_env'] or 0)
            
            # Compute derived
            vente_theo_ttc = max(0, si_ttc + achats_ttc - sf_ttc - demarque_ttc)
            marge_ttc = ca_ttc - vente_theo_ttc
            marge_pct = (marge_ttc / vente_theo_ttc * 100.0) if vente_theo_ttc > 0 else 0.0
            
            result.append({
                'sous_categorie': cat,
                'si_ttc': si_ttc,
                'achats_ttc': achats_ttc,
                'sf_ttc': sf_ttc,
                'demarque_ttc': demarque_ttc,
                'vente_theo_ttc': vente_theo_ttc,
                'marge_ttc': marge_ttc,
                'marge_percent': round(marge_pct, 2)
            })
        
        return result
```

**Clarify semantics**: For a date range report:
- `SI` = stock value at beginning of first day in range (from `Tcollecte.si` of that first day)
- `SF` = stock value at end of last day in range (from `Tcollecte.sf` of that last day)
- `Achats` = sum of all `achats` across days in range
- `Démarque` = sum of `env` across range
- `Vente_theorique` = SI + Achats - SF - Démarque (computed once using aggregated)
- `Marge` = Σ daily margins OR CA - Vente_theorique (use final CA)

We'll implement proper aggregation in `AnalyseJournaliereService` using the reporting table.

#### 3.4 NFR Report Integration

`get_nfr_report(year, month)` will query `Tsf` for all days in the month:

```python
def get_nfr_report(self, year: int, month: int) -> List[dict]:
    from datetime import date, timedelta
    
    # Determine date range for month
    start = date(year, month, 1)
    if month == 12:
        end = date(year + 1, 1, 1) - timedelta(days=1)
    else:
        end = date(year, month + 1, 1) - timedelta(days=1)
    start_str = start.strftime("%Y-%m-%d")
    end_str = end.strftime("%Y-%m-%d")
    
    with self.repo._connect() as conn:
        rows = conn.execute("""
            SELECT 
                c.nom AS categorie,
                SUM(t.ca_ttc) AS ca_ttc,
                SUM(t.achats_ttc) AS achats_ttc_total,
                SUM(t.marge_ttc) AS marge_ttc,
                AVG(t.marge_percent) AS avg_marge_pct
            FROM Tsf t
            JOIN categories c ON t.categorie_id = c.id
            WHERE t.jour >= ? AND t.jour <= ?
            GROUP BY c.nom
            ORDER BY c.nom
        """, (start_str, end_str)).fetchall()
        
        result = []
        for r in rows:
            ca_ttc = int(r['ca_ttc'] or 0)
            ca_ht = int(round(ca_ttc / 1.2))
            marge_ttc = int(r['marge_ttc'] or 0)
            # NFR% = (Marge TTC / CA TTC) × 100
            nfr_pct = (marge_ttc / ca_ttc * 100.0) if ca_ttc > 0 else 0.0
            
            result.append({
                'categorie': str(r['categorie']),
                'ca_ttc': ca_ttc,
                'ca_ht': ca_ht,
                'marge': marge_ttc,
                'nfr_percent': round(nfr_pct, 2)
            })
        return result
```

**Key decision**: NFR uses **total monthly margin ÷ total monthly CA** (not average of daily margin %). This matches standard accounting practice.

```python
def initialize_day(self, jour: str, carry_si_from: Optional[str] = None) -> None:
    """
    Initialize Tcollecte rows for all OW categories.
    
    Rules:
    - If carry_si_from provided: SI = that date's SF from Tcollecte (must exist and be closed)
    - Else (first day ever): SI = current total stock value from produits table
    - Other fields (achats, ca_live, ca_final, sf, env) = 0
    - cloturee = 0
    - Idempotent: existing rows are updated only if si was 0 (safeguard)
    """
    if carry_si_from:
        # Carries forward SI from previous day's SF
        prev_tracks = self.list_tracks(carry_si_from)
        if not prev_tracks:
            raise ValueError(f"No tracking data found for previous day {carry_si_from}")
        si_by_cat = {t.category_name: t.stock_final for t in prev_tracks}
    else:
        # First run: compute SI from current produits snapshot
        with self._connect() as conn:
            rows = conn.execute("""
                SELECT c.nom AS categorie, SUM((p.stock_boutique + p.stock_reserve) * p.pa) AS si
                FROM produits p
                JOIN categories c ON p.categorie_id = c.id
                GROUP BY c.nom
            """).fetchall()
            si_by_cat = {str(r['categorie']): int(r['si'] or 0) for r in rows}
    
    # Resolve category IDs
    cat_names = list(si_by_cat.keys())
    cat_id_map = self._resolve_categories(cat_names)
    
    # Upsert rows
    with self._connect() as conn:
        for cat_name, si_val in si_by_cat.items():
            cat_id = cat_id_map.get(cat_name)
            if not cat_id:
                continue
            conn.execute("""
                INSERT INTO Tcollecte (
                    jour, categorie_id, si, achats, ca, ca_temporaire,
                    sf, env, vente_theorique, marge, cloturee, updated_at
                ) VALUES (?, ?, ?, 0, 0, 0, 0, 0, 0, 0, 0, datetime('now'))
                ON CONFLICT(jour, categorie_id) DO UPDATE SET
                    si = excluded.si,
                    updated_at = datetime('now')
                WHERE Tcollecte.si = 0  -- Only set SI if not already set
            """, (jour, cat_id, si_val))
        conn.commit()
```

#### 6.2 SF Update (`update_stock_metrics`)

```python
def update_stock_metrics(self, jour: str) -> None:
    """
    Update SF and ENV from source tables.
    
    SF = current produits stock value (sum of (boutique+reserve)*pa per category)
    ENV = sum of removed product values from historique_produits_enleves for the day.
    """
    with self._connect() as conn:
        # SF: current product stock snapshot per category
        sf_rows = conn.execute("""
            SELECT c.nom AS categorie, SUM((p.stock_boutique + p.stock_reserve) * p.pa) AS sf_ttc
            FROM produits p
            JOIN categories c ON p.categorie_id = c.id
            GROUP BY c.nom
        """).fetchall()
        sf_map = {str(r['categorie']): int(r['sf_ttc'] or 0) for r in sf_rows}
        
        # ENV: today's removals
        env_rows = conn.execute("""
            SELECT categorie, SUM(valeur) AS env_ttc
            FROM historique_produits_enleves
            WHERE jour = ?
            GROUP BY categorie
        """, (jour,)).fetchall()
        env_map = {str(r['categorie']): int(r['env_ttc'] or 0) for r in env_rows}
        
        # Update each category
        updates = []
        for categorie, sf_val in sf_map.items():
            env_val = env_map.get(categorie, 0)
            cat_id_row = conn.execute("SELECT id FROM categories WHERE nom = ?", (categorie,)).fetchone()
            if cat_id_row:
                updates.append((sf_val, env_val, jour, cat_id_row['id']))
        
        if updates:
            conn.executemany("""
                UPDATE Tcollecte
                SET sf = ?, env = ?, updated_at = datetime('now')
                WHERE jour = ? AND categorie_id = ?
            """, updates)
        
        conn.commit()
    
    # Recompute derived fields after SF/ENV change
    self.recompute_derived_fields(jour)
```

#### 6.6 Reporting Refresh (Tsf)

```python
def refresh_tsf(self, start_date: str, end_date: str) -> None:
    """Refresh Tsf table for date range from Tcollecte + current produits stock."""
    with self._connect() as conn:
        # Get Tcollecte rows in range
        rows = conn.execute("""
            SELECT 
                t.jour, t.categorie_id, c.nom AS categorie,
                t.si, t.achats, t.ca, t.ca_temporaire, t.env, t.cloturee
            FROM Tcollecte t
            JOIN categories c ON t.categorie_id = c.id
            WHERE t.jour >= ? AND t.jour <= ?
        """, (start_date, end_date)).fetchall()
        
        if not rows:
            return
        
        # Compute current SF snapshot per category from produits (live stock)
        sf_rows = conn.execute("""
            SELECT c.nom AS categorie, SUM((p.stock_boutique + p.stock_reserve) * p.pa) AS sf_ttc
            FROM produits p
            JOIN categories c ON p.categorie_id = c.id
            GROUP BY c.nom
        """).fetchall()
        sf_map = {str(r['categorie']): int(r['sf_ttc'] or 0) for r in sf_rows}
        
        # Prepare Tsf inserts
        tsf_data = []
        for r in rows:
            jour = r['jour']
            categorie_id = r['categorie_id']
            categorie = str(r['categorie'])
            is_closed = r['cloturee'] == 1
            
            si_ttc = int(r['si'] or 0)
            achats_ttc = int(r['achats'] or 0)
            env_ttc = int(r['env'] or 0)
            ca_ttc = int(r['ca'] if is_closed else r['ca_temporaire'] or 0)
            sf_ttc = sf_map.get(categorie, 0)
            
            # Derived
            vente_theo_ttc = max(0, si_ttc + achats_ttc - sf_ttc - env_ttc)
            marge_ttc = ca_ttc - vente_theo_ttc
            marge_pct = (marge_ttc / vente_theo_ttc * 100.0) if vente_theo_ttc > 0 else 0.0
            
            tsf_data.append((
                jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                sf_ttc, vente_theo_ttc, marge_ttc, marge_pct, int(is_closed)
            ))
        
        # Clear existing for range and insert fresh
        conn.execute("DELETE FROM Tsf WHERE jour >= ? AND jour <= ?", (start_date, end_date))
        conn.executemany("""
            INSERT INTO Tsf (
                jour, categorie_id, si_ttc, achats_ttc, ca_ttc, env_ttc,
                sf_ttc, vente_theorique_ttc, marge_ttc, marge_percent, is_closed, refreshed_at
            ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, datetime('now'))
        """, tsf_data)
        conn.commit()
```

## Questions / Open Issues

1. **Tsf refresh frequency**: Should `refresh_tsf()` be called automatically:
   - After every `update_live_ca`? (May be frequent; could batch every N minutes)
   - On-demand only when report dialog opens? (Simpler; slight delay on open)
   I recommend **on-demand + nightly batch**. On report open, if Tsf data is older than 1 hour, auto-refresh range.

3. **Multi-category sales**: Sale line items may have category assigned; product→category resolution happens at sale recording. Ensure `category_name` used in `increments` matches category names in DB (use product's category at time of sale, not current product category — if product recategorized, historical sales should use old category). Consider storing category_id in ventes line? Current `ventes` table only stores `produit_nom`, not category. Category resolution happens at display time by looking up current product category. This could cause mis-attribution if product category changes. Should we archive category at sale time? **This is out of scope for Tcollecte rewrite but note as risk.** For now, assume categories stable.

4. **Time zones**: All dates stored as `YYYY-MM-DD` strings without timezone. App runs in local time. No DST issues.
